¿Qué es entonces una base de datos - DB?

Es un conjunto de archivos que contienen información de manera organizada, consistente y que se relaciona entre si.

¿En qué casos es conveniente usar bases de datos?

Es principalmente mas útil cuando se tiene una gran cantidad de información/registros, que seria prácticamente imposible revisarlos manualmente, también cuando se requiere proteger ciertos datos y para que sea mas fácil añadir información.

¿Qué es un sistema de gestión de base de datos - DBMS?

Es un software de interfaz que se encuentra entre el usuario, la base de datos y los programas que lo utilizan, para que sea mas facil y conveniente de utilizar

Enlista y explica las funciones/responsabilidades que tienen los DBMS.
  • Redundancia de datos: El sistema al tener los ficheros integrados, evita que se guarden varias copias de un mismo archivo, reduciendo el espacio de almacenamiento total y el ruido de tener varias veces la misma información.
  • Consistencia de datos: Al controlar la redundancia ayuda a que la consistencia se mantenga, ya que toda la información se actualiza y no se tienen datos obsoletos.
  • Compartir datos: Sin una base de datos, los ficheros solo se pueden almacenar y modificar una a la vez, con los DBMS permiten que varias personas accedan a estos datos.
  • Seguridad: Al tener limitaciones especificando que usuarios pueden acceder a que información ayuda a proteger datos de externos o de ser modificados o accedidos por un usuario que no tendría que tener acceso.
  • Accesibilidad de los datos: No es necesario tener conocimientos de programación o saber exactamente como se almacenan los datos para poder leer la información.
Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

No necesariamente proyecto, pero cuando tuve que ayudar en la parte administrativa de una escuela, hubiera facilitado mucho el trabajo tener una base de datos antes que tener que obtener la información de diferentes fuentes, internet, archivos en varias carpetas y físicos.
