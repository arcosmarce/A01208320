¿Cuál es la diferencia entre Internet y la World Wide Web?

Aunque relacionados, no son lo mismo, el internet es la infraestructura que le da soporte al world wide web, que ofrece el servicio de información a la que se puede acceder.

https://www.comunicatefacil.com/diferencias-world-wide-web-internet/
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET: Se pide la recuperación de un recurso.
HEAD: Es similar a GET, pero sin el cuerpo de la respuesta.
POST: Usado para enviar un recurso, puede causar un cambio de estado o efectos en el servidor.
PUT: Reemplaza todas las representaciones actuales del recurso de destino con la carga útil de petición.
PATCH: Modifica parcialmente un recurso.
DELETE: Borra el recurso indicado.

https://developer.mozilla.org/es/docs/Web/HTTP/Methods

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Si todo se realiza de manera correcta, es responsabilidad, por que dicho código reporta que no se pudo acceder a algún recurso, por lo que el sitio tiene entonces un enlace o referencia roto o dañado que se tiene que reparar.
